# 方法评估：当前的全身力感知路线能否达成"盲走迷宫"目标？

> 结论先行：**方法选型是对的，不需要换路；当前实现距目标有三个明确缺口，
> 每个缺口都有文献验证过的修复方案。** 唯一没被任何人证明过的核心科学风险
> （行走中的持续贴墙信号）可以用 1 天的桥接实验检验。

> 依据：`docs/blind_maze_research_and_method.md`（9 路深度调研）+ Plan A v3 实测数据。

## 1. 现状 vs 任务需求（拿数字说话）

| 指标 | 我们现状 (v3, 静态+瞬时力) | 迷宫任务需求 (调研得出) | 判定 |
|---|---|---|---|
| 检测召回 | 0.86 | 去抖后 >0.85 | ✅ 已达标 |
| 误报率 | ~55% (prec 0.45) | 去抖后 <2% | ⚠️ 可修：k=3 帧确认可把 15%→0.3%（文献）；长窗+去抖大概率够 |
| 力方向 | cos 0.35 (≈70° 误差) | 60° 扇区 (±30°) | ❌ 不达标 → H=50 可修（SixthSense 同任务 17–29°） |
| 定位 | 5 解剖区域 acc 0.48 | 6 扇区（机器人系方位） | ✅ 换标签后变容易：扇区由方向+方位决定；前向接触多落在手臂（我们最强 0.7） |
| 工况 | 静态站立 | 行走中 + 持续贴墙 | ❌ 最大缺口（风险 1） |
| 实时性 | MLP 微秒级 | 闭环控制 | ✅ |

**关键认知**：导航层把感知及格线拉得很低——6 扇区二值 + 去抖就够跑可证明的
Pledge 迷宫逃逸。我们不是造完美力传感器，是造"够导航用的粗接触罗盘"。

## 2. 为什么不需要换方法

1. **方法族已被真机验证**：SixthSense 用同路线（仿真训练、纯本体、q+dq+tau_est+IMU）
   在真机 G1 行走中做到检测 85.5% / 误报 1.9%。这条路有人走通过。
2. **"其他办法"已是我们方案的组件而非替代品**：动量观测器→备用输入通道；
   NEXT 电流残差→真机自校准桥；隐式策略适应→上游 GentleHumanoid 框架已有。
3. **缺口↔文献配方一一对应**：W6→H50（四篇独立工作收敛值）；静态→行走数据；
   瞬时→持续接触；解剖区域→扇区标签；+去抖。没有一个缺口需要发明新东西。

## 3. 两个真正未被证明的风险

**风险 1（核心科学风险）：行走步态噪声可能淹没持续贴墙信号。**
物理上有两个可抓信号：撞墙瞬间的冲击瞬态（成熟可抓，带通滤波）+
贴墙期的恒定力矩偏置（长窗/残差可见，但**没人测过**）。
→ Phase 0a 桥接实验用 1 天直接回答。

**风险 2：sim-to-real。** tau_est 噪声模型未知 + 踝并联机构仿真未建模。
缓解链已备：域随机化四件套 + 噪声注入（仿真侧）、NEXT 10 分钟自校准（真机侧）、
掩踝通道。有先例无保证。

## 4. 兜底链（最终能成的底气）

Gate 0 不过时按序启用，一层比一层保守：
1. **动量残差通道**（物理先验，MOB-Net 证明 GRU 修正后好 3–5×）；
2. **手臂触须姿态**（把接触引到定位最准的部位 0.7）；
3. **降速行走**（噪声随动态度下降）；
4. **终极兜底："停-感-走"步态**——走几步、短暂停顿感知、继续。停顿时=静态工况，
   而静态感知我们已用实数据证明可行（0.86 召回）。迷宫任务允许机器人自己选
   什么时候感知（不同于推倒恢复必须动态感知）→ **这条几乎保证仿真闭环能通，
   代价只是走得慢**。

## 5. 概率估计（诚实）

| 里程碑 | 概率 | 依据 |
|---|---|---|
| 仿真闭环走出迷宫 (Phase 0–2) | **~80–85%** | 缺口修复全有先例 + 停-感-走保底 |
| 真机走出迷宫 (Phase 3–4) | **~50–60%** | 持续接触估计的 sim-to-real 无人做过=研究前沿；失败模式是降级（慢/触须）非归零 |

## 6. 判定

**不换方法。按既定计划执行**：0a 桥接（1 天，检验风险 1）→ 0b 真墙数据 →
1′ 迷宫+导航（真值接触）→ 2′ 闭环 → 3′ 真机准备。
训练清单见对话记录（T0–T9），代码路线见 `docs/blind_maze_research_and_method.md` Part III。
